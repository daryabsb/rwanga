Spacing & Typography
Spacing Scale
All spacing via BS5 utilities — p-*, m-*, gap-*. Never write custom margin/padding CSS. The platform uses generous spacing: content breathes on TV/smartboard, stays compact on mobile. Use the tighter end of the scale for dense data tables; the looser end for reading/review views.

Spacing & Typography
Line Height & Text Rhythm
Kurdish script (Noto Sans Arabic) needs more generous line-height than Latin. Tight line-height breaks readability in RTL. Use these values via inline style or BS5 lh-* utilities. Never compress line-height below 1.5 for body text.
