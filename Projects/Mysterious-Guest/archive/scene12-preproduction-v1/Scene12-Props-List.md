## Overview

Scene 12 is the most prop-intensive scene in the test shoot. The garden house interior must feel like a theater artist's living museum — cluttered, artistic, slightly unsettling. Props divide into two categories: items already present in the location, and items that must be sourced or fabricated.

---

## CATEGORY A: Hero Props (On Camera, Story-Critical)

These props are referenced in dialogue, seen in close-up, or carry narrative weight. They MUST be correct.

### A1. Taxidermy Deer Head (Mounted)

| Field | Detail |
|-------|--------|
| Priority | CRITICAL — appears in 6 shots (12.2, 12.5, 12.6, 12.13, 12.14, 12.15) |
| Description | Mounted deer head with antlers, wall-hung. Must look realistic enough for close-up (Shot 12.5 is a slow push directly toward it). |
| Size | Life-size. Antler spread approximately 60–80cm. |
| Placement | Center of north (back) wall, above the curtains. Height: ~2m from floor (high enough to look down at the room). |
| Critical Detail | Glass eyes must be present and reflective — a practical light will be positioned to create a pinpoint GLINT in one eye. This is a story beat: "Look at those eyes, how alive they are." |
| Source Options | 1) Real taxidermy (borrow from hunter, antique shop, or prop house). 2) High-quality replica (theatrical supply). 3) Custom fabrication (sculptor — allow 2+ weeks lead time). |
| Status | [ ] Sourced / [ ] On order / [ ] Not yet |
| Backup Plan | If unavailable: a large, realistic painting or photograph of a deer in a heavy frame. Loses impact but preserves the scene. |
| Notes | Confirm mounting hardware — the wall must support the weight. Test mount before shoot day. If the location wall is fragile, bring a free-standing mount/bracket system. |

### A2. Theater Masks (3–4 pieces)

| Field | Detail |
|-------|--------|
| Priority | CRITICAL — appears in 4 shots (12.2, 12.7, 12.8, 12.9), has dedicated insert (12.8) |
| Description | Pale white/cream expressionless masks. Not colorful carnival masks — these are blank, neutral theater masks with hollow dark eye sockets. Unsettling, not decorative. Think commedia dell'arte neutral masks, or plain white drama masks. |
| Quantity | 3 minimum, 4 preferred. One larger (center), two flanking. |
| Size | Face-size. Approximately 20–25cm tall each. |
| Placement | West wall, near the bookcase area. Partially in shadow. Hung in a loose cluster — one higher, two lower. NOT symmetrically arranged — they should feel casually placed, as if someone just hung them and forgot. |
| Material | Hard shell (plaster, resin, or papier-mâché). Must hold shape and catch light cleanly. No fabric masks. |
| Critical Detail | The eye sockets must be DEEP enough to hold shadow. When lit from below (Shot 12.8), the eyes should become black voids. |
| Source Options | 1) Theatrical supply store. 2) Art supply store (blank mask forms). 3) Handmade — papier-mâché over form, painted matte white/cream. Allow 3–5 days for fabrication + drying. |
| Status | [ ] Sourced / [ ] On order / [ ] Not yet |
| Notes | Bring hanging hardware (small nails or hooks + fishing line for invisible mount). Test arrangement on location wall before shoot day. |

### A3. "Stories Are Waiting" Paper

| Field | Detail |
|-------|--------|
| Priority | HIGH — close-up insert (Shot 12.11), referenced in dialogue |
| Description | A piece of paper with the phrase "Stories are waiting" written on it. This is supposedly the title of Nali's new play, discovered by Gesha on his desk. |
| Size | Standard A4 or slightly smaller (a torn/trimmed page feels more artistic than a clean sheet). |
| Paper Type | Cream or off-white, slightly textured. Not bright white printer paper. Something that feels handled. |
| Writing | Handwritten in dark ink (black or dark brown). The handwriting should look like a writer's — confident but slightly hurried. Not calligraphy, not messy. As if someone wrote it during a thought and set it down. |
| Language | Confirm with director: English? Kurdish? Both? The screenplay has it in Kurdish: "چیرۆکەکان لە چاوەڕوانیدان" |
| Placement | On the desk, among other scattered papers. Not centered or prominent — Gesha discovers it, she doesn't find it placed for her. |
| Status | [ ] Prepared / [ ] Not yet |
| Notes | Prepare 3–4 versions with slightly different handwriting quality. The DP will choose which reads best on camera. Also prepare the surrounding papers — they should have fragments of writing, sketches, or notes to sell the "working desk" look. |

## CATEGORY B: Set Dressing Props (Visible, Atmosphere)

These props are seen on camera and contribute to the world, but are not referenced in dialogue. They must look right but don't need to be hero-quality.

### B1. Wall Weapons (Daggers + Sword)

| Field | Detail |
|-------|--------|
| Priority | MEDIUM — visible in wide shots (12.2, backgrounds) |
| Description | Decorative edged weapons mounted on the west wall. The screenplay mentions daggers and a curved sword. These sell the idea that Nali collects strange objects. |
| Items | 2–3 daggers (ornamental, Kurdish/Middle Eastern style preferred). 1 curved sword or shamshir. |
| Placement | West wall, spread across ~1m of wall space, between the photo clusters. |
| Source Options | 1) Antique/souvenir shops in Sulaymaniyah bazaar. 2) Theatrical prop weapons (safer on set). 3) Borrow from private collections. |
| Status | [ ] Sourced / [ ] Not yet |
| Safety Note | If using real blades, designate a props master to handle. Keep sheathed until mounted. Inform all crew of blade locations. If using theatrical props, no safety concern. |

### B2. Abstract/Alien-like Sculptures

| Field | Detail |
|-------|--------|
| Priority | MEDIUM — visible on shelves in wide shots |
| Description | Small strange sculptures mixed in with the existing pottery and glassware on the east wall shelves. They should feel out of place — objects that don't belong in a normal house. |
| Items | 3–5 small sculptures. Could be: abstract ceramic forms, carved wooden figures, found-object assemblages, stone carvings. Anything that looks like it has a story but you can't quite tell what it is. |
| Size | Small enough to sit on shelves — 10–25cm each. |
| Source Options | 1) Art school student work (cheap, genuinely strange). 2) Flea market/antique bazaar finds. 3) Fabricate from clay/plaster — quick and cheap. |
| Status | [ ] Sourced / [ ] Not yet |
| Notes | These don't need to be beautiful. They need to be WEIRD. The weirder the better — they contribute to the feeling that Nali's house is a collection of obsessions. |

## CATEGORY C: Existing Location Items (Verify & Dress)

These items are already present in the location. They need to be verified, cleaned, and positioned for camera.

| Item | Current Status | Action Needed |
|------|---------------|---------------|
| Wooden dining table + white chairs | Present | Clean table surface. Remove any modern items (phone chargers, etc). Position chairs evenly. |
| Red Persian rugs | Present | Straighten. Remove any stains or debris. Ensure they don't bunch during dolly movement. |
| Wooden shelving (east wall) with pottery/glassware | Present | Clean glass items. Add sculptures (B2) among existing items. Remove any obviously modern items (plastic containers, etc). |
| Metal display shelf with dishes | Present | Clean. Can stay as-is — adds to cluttered feel. |
| Pendant lamps (2–3) | Present | Confirm all working. Put on dimmers. Replace any dead bulbs. Warm-tone bulbs only (2700K–3000K). |
| Sofa with colorful cushions (west wall) | Present | Straighten cushions. Arrange patchwork fabrics artistically. The sofa should look inviting but eccentric. |
| Bookcase with books | Present | Can stay as-is. Adds intellectual character. |
| Curtains (back wall) | Present | Close most of the way — leave a gap for faked "evening light" from outside. Steam or iron if wrinkled. |
| Photos/frames on west wall | Present | Keep — adds personal history to the space. Add daggers (B1) between them. |
| Large painted canvas (near doors) | Present | Keep. Reposition if it blocks camera angles toward the doors. |
| TV / monitor (west wall) | Present | TURN OFF. Cover screen if reflective. A powered-off TV in a dark room creates a dead spot on camera — consider draping a cloth over it or removing it entirely. |
